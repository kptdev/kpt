<!--mdtogo:Short
    Displays OSS license information.
-->

`info` displays license information of the OSS libraries used by `kpt`.

### Synopsis

<!--mdtogo:Long-->

```sh
kpt alpha license info
```

<!--mdtogo-->

### Examples

<!--mdtogo:Examples-->

```shell
# display license information for OSS libraries.
$ kpt alpha license info | less
```
